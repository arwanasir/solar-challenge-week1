# solar-challenge-week1

## Project Overview

Analysis of solar farm data from Benin, Sierra Leone, and Togo to identify high-potential regions for solar investments.

## Environment Setup

1. Clone this repository
2. Create virtual environment: `python -m venv .venv`
3. Activate: `.venv\Scripts\activate` (Windows) or `source .venv/bin/activate` (Mac/Linux)
4. Install dependencies: `pip install -r requirements.txt`

## Project Structure

- `notebooks/` - Jupyter notebooks for EDA of each country
- `data/` - Solar measurement datasets (local only, not committed)
- `src/` - Source code modules
- `tests/` - Test files

## Analysis Completed

- Data profiling, cleaning, and exploratory analysis
- Time series analysis and pattern detection
- Correlation analysis and visualization
- Cross-country comparison
